== Description ==
Review Gate is a lightweight WordPress plugin designed to help capture more 5-star leads by prompting users to leave a review based on their experience. With a simple and intuitive review step interface and an emoji-based rating system, this plugin improves user engagement and streamlines the review process for your business.

== Installation ==
1. Upload the entire `dk-review-gate` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings as needed from the WordPress admin dashboard "Settings > DK Review Gate".
4. Insert the `[dk_review_gate]` shortcode into any post or page where you want the review gate to appear.
